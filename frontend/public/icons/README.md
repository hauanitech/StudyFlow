# PWA Icons for StudyFlow

This folder should contain the following icon files for the Progressive Web App:

## Required Icons

| Filename | Size | Purpose |
|----------|------|---------|
| `icon-72x72.png` | 72x72 | Small devices |
| `icon-96x96.png` | 96x96 | App shortcuts |
| `icon-128x128.png` | 128x128 | Medium devices |
| `icon-144x144.png` | 144x144 | Android splash |
| `icon-152x152.png` | 152x152 | iOS home screen |
| `icon-192x192.png` | 192x192 | Android home screen (required) |
| `icon-384x384.png` | 384x384 | Large displays |
| `icon-512x512.png` | 512x512 | Splash screen (required) |

## Icon Guidelines

1. **Format**: PNG with transparency
2. **Style**: Should work on both light and dark backgrounds
3. **Maskable**: Include safe zone (inner 80%) for adaptive icons on Android
4. **Design**: Simple, recognizable at small sizes

## Quick Generation

You can generate all sizes from a single 512x512 source using:
- [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
