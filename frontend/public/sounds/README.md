# Pomodoro Timer Sound Files

This folder should contain the following audio files for the Pomodoro timer notifications:

## Required Audio Files

| Filename | Description | Recommended Duration |
|----------|-------------|---------------------|
| `chime.mp3` | Gentle chime sound | 1-2 seconds |
| `bell.mp3` | Classic bell notification | 1-2 seconds |
| `gong.mp3` | Deep meditation gong | 2-3 seconds |
| `birds.mp3` | Pleasant bird chirping | 1-2 seconds |
| `piano.mp3` | Elegant piano chord | 1-2 seconds |
| `digital.mp3` | Modern digital alarm | 1-2 seconds |

## Audio File Guidelines

1. **Format**: MP3 format is recommended for best browser compatibility
2. **Duration**: Keep sounds short (1-3 seconds) for notifications
3. **Volume**: Normalize audio levels to prevent sudden loud sounds
4. **Quality**: 128kbps or higher for good sound quality
5. **File size**: Keep files under 100KB each for quick loading

## Free Sound Resources

You can find royalty-free notification sounds at:
- [Freesound.org](https://freesound.org/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [Zapsplat.com](https://www.zapsplat.com/)

## Note

The default beep sound is generated using the Web Audio API and doesn't require a file.
Unauthenticated users will only have access to the default beep and "no sound" options.
